HR Analytics Dashboard (Power BI)
Overview

This project is an HR Analytics Dashboard built using Power BI to analyze employee data and generate insights on attrition, demographics, and workforce trends.

Features
Employee attrition analysis
Gender and age distribution insights
Department-wise performance overview
Salary and employee satisfaction trends
Interactive filters and slicers for dynamic exploration
Tools Used
Power BI Desktop
Excel / SQL (data source, if applicable)
Data modeling and DAX measures
Dashboard Preview

Project File
HR Dashboard.pbix → Open using Power BI Desktop
How to Use
Download the repository
Open the .pbix file in Power BI Desktop
Explore interactive visuals using filters and slicers
